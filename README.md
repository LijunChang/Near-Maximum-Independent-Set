Computing A Near-Maximum Independent Set in Linear Time by Reducing-Peeling

# Compile
```sh
$ make
```

# Usage
```sh
./mis {alg} {graph_directory}
```
alg is chosen from "greedy, greedy_dynamic, BDOne, BDTwo, LinearTime, NearLinear"
